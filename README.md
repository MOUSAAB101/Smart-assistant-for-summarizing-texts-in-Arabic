# Smart-assistant-for-summarizing-texts-in-Arabic
# Arabic Text Summarization Project
ملاحظة هذه ليست النسخة النهائية ممكن ان يتم التعديل على الاطروحة والعرض التقديمي!!!!!!!!!!!

هذا المشروع يحتوي على نموذج AraT5v2 لتلخيص النصوص العربية.  
يتضمن:
- كود ليس النهائي اذا  طلب اي تعديل او تغيير على الكود `Smart-assistant-for-summarizing-texts-in-Arabic.ipynb` 
- ملف الأطروحة `Smart assistant for summarizing texts in Arabic document`
- مثال لتلخيص ملفات PDF من الجهاز

## كيفية الاستخدام

1. افتح `Smart-assistant-for-summarizing-texts-in-Arabic.ipynb` على Google Colab
2.تثبيت المكتبات المطلوبة: `!pip install -r requirements.txt`
3. استخدم الدوال:
   - `summarize_arabic(text)` لتلخيص نص
   - `summarize_pdf_from_upload(file.pdf)` لتلخيص PDFمحلي
4. النتائج تعرض مباشرة على Notebook
